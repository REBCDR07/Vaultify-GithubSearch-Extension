Create a Next.js 14 App Router project that is a pixel-perfect interactive mockup 
of a Chrome Extension popup UI called "RepoRadar".
This is a FRONTEND ONLY prototype — hardcoded mock data, no real API calls.

## TECH STACK
- Next.js 14 App Router
- TypeScript
- Tailwind CSS (for layout utilities only)
- Framer Motion (for animations)
- NO external UI libraries (shadcn, radix, etc.)
- All custom CSS via CSS Modules or a global CSS file

## FONTS (next/font or Google Fonts)
- Unbounded — brand/display
- Rajdhani — body/UI text
- Oswald — labels, badges, metadata

## DESIGN TOKENS (globals.css CSS variables)
--bg: #0d1117
--surface: #161b22
--surface-2: #1c2128
--border: #30363d
--border-subtle: #21262d
--blue: #58a6ff
--green: #3fb950
--orange: #e3b341
--red: #f78166
--text: #e6edf3
--muted: #8b949e
--subtle: #484f58

Custom scrollbar: 4px, dark, rounded
Micro-transitions: 150ms ease on all interactive elements
Aesthetic: flat, dense, technical — NO gradients, NO shadows, NO rounded excess

## COMPONENT SPECS

### PopupShell.tsx
- 'use client'
- Fixed container: width 420px, height 600px, overflow hidden
- background: var(--bg), color: var(--text)
- Centered on page with margin auto, border 1px var(--border)
- Manages global state: searchState, searchQuery, favoritedRepos (Set<number>), 
  showBanner, showFilters, activeCategory, filterState, loadingSteps
- Pass down via props (no Context needed at this scale)

### Header.tsx
Left side:
- RadarIcon SVG (20px): circle + search line + green cross inside
- "REPO" + "RADAR" (RADAR in --blue) — Unbounded font 11px
Right side:
- HeartIcon button → opens favorites page (for now just console.log)
- GearIcon button → toggles NotConfiguredBanner visibility
Both buttons: 28px square, --surface-2 bg on hover, 1px --border on hover, transition

### SearchBar.tsx
Props: query, onQueryChange, onSearch, showFilters, onToggleFilters, filterState, onFilterChange
- Input: full width, --surface bg, 1px --border, 8px padding-left for icon
- Focus: border-color var(--blue)
- SearchIcon: absolute left, 14px, --muted color
- Clear button (×): absolute right, visible only when query.length > 0
- "Filtres" toggle button: Oswald 10px, sliders icon, active count badge (--blue bg)
- Filter panel (AnimatePresence + motion.div slide down):
  2-column grid, --surface bg, 1px --border, 8px radius
  4 selects: Langage, Stars min, Mise à jour, Licence
  "Réinitialiser" button: hover red border and text

### CategoryPills.tsx  
Props: activeCategory, onSelect
Categories with their search queries:
- "UI Libs" → "composants UI React modernes"
- "Icons" → "icônes SVG légères open source"
- "CLI Tools" → "outils CLI développeur productivité"
- "CSS" → "frameworks CSS modernes"
- "State" → "state management léger javascript"
- "Anim" → "animation javascript bibliothèque"

Pill styles:
- Default: --surface-2 bg, --border-subtle border, --muted text, Oswald 10px uppercase
- Active/hover: rgba(--blue, 0.08) bg, --blue border, --blue text

### ResultsArea.tsx
Props: state, repos, favoritedRepos, onToggleFav, suggestions
Renders one of 4 states:

IDLE state:
- Centered vertically: large RadarIcon (48px, muted) + "Cherche des repositories" (Rajdhani 600 15px muted) + subtitle (12px subtle)

LOADING state:
- 4 LoadingStep items stacked vertically, gap 10px
- Each step: dot (8px circle) + label (Oswald 12px)
- Dot states: gray (pending) → pulsing blue glow (active) → solid green (done)
- Active dot: CSS animation pulse (scale 1→1.3→1, 1s infinite)
- Steps activate sequentially via useEffect with 1200ms intervals

RESULTS state:
- Results header: "10 résultats" (Oswald 11px muted) + "cache" badge (orange outlined, 9px) + "~1847 tokens" (subtle right)
- RepoCard list with staggered Framer Motion entrance
- SuggestionsRow at bottom

ERROR state:
- Red warning icon + error message centered

### RepoCard.tsx
Props: repo, isFavorited, onToggleFav, index (for animation delay)

Framer Motion: initial {opacity:0, y:4} animate {opacity:1, y:0} transition {delay: index * 0.04}
